# All APIs Report

## Cafe Routes (`server/routes/cafe.routes.js`)
- `POST /api/cafe/register` (Controller: `registerCafe`, Middleware: `upload.single('profilePicture')`)
- `POST /api/cafe/login` (Controller: `login`)
- `GET /api/cafe/status` (Controller: `checkStatus`, Middleware: `verifyToken`)
- `PUT /api/cafe/update` (Controller: `updateCafe`, Middleware: `verifyToken`)

## Auth Routes (`server/routes/auth.routes.js`)
- `POST /api/auth/google` (Controller: `googleAuth`)
- `GET /api/auth/me` (Controller: `getMe`, Middleware: `verifyToken`)

## Order Routes (`server/routes/order.routes.js`)
- `GET /api/orders` (Controller: `getOrders`, Middleware: `verifyToken`)
- `POST /api/orders/create` (Controller: `createOrder`)
- `PUT /api/orders/:id/status` (Controller: `updateOrderStatus`, Middleware: `verifyToken`)

## Admin Routes (`server/routes/admin.routes.js`)
- `GET /api/admin/pending-cafes` (Controller: `getPendingCafes`, Middleware: `verifyToken`)
- `PUT /api/admin/approve-cafe/:id` (Controller: `approveCafe`, Middleware: `verifyToken`)

## Event Routes (`server/routes/event.routes.js`)
- `POST /api/events/upload-banner` (Controller: `uploadBanner`, Middleware: `upload.single('banner')`)
- `POST /api/events/create` (Controller: `createEvent`, Middleware: `verifyToken`)
- `POST /api/events/update/:id` (Controller: `updateEvent`, Middleware: `upload.single('banner')`, `verifyToken`)
- `DELETE /api/events/delete/:id` (Controller: `deleteEvent`, Middleware: `verifyToken`)
- `GET /api/events/all` (Controller: `getAllEvents`)
- `GET /api/events/my-events` (Controller: `getMyEvents`, Middleware: `verifyToken`)
- `GET /api/events/dashboard` (Controller: `getDashboardStats`, Middleware: `verifyToken`)
- `GET /api/events/earnings` (Controller: `getEarningsStats`, Middleware: `verifyToken`)
- `GET /api/events/analytics/:eventId` (Controller: `getEventAnalytics`, Middleware: `verifyToken`)
- `GET /api/events/:id` (Controller: `getEventById`)
- `POST /api/events/save-draft` (Controller: `saveDraft`, Middleware: `verifyToken`)
- `POST /api/events/publish/:id` (Controller: `publishEvent`, Middleware: `verifyToken`)
- `POST /api/events/unpublish/:id` (Controller: `unpublishEvent`, Middleware: `verifyToken`)
- `POST /api/events/cancel/:id` (Controller: `cancelEvent`, Middleware: `verifyToken`)
- `POST /api/events/register` (Controller: `registerEvent`)
- `POST /api/events/verify-ticket` (Controller: `verifyTicket`, Middleware: `verifyToken`)
